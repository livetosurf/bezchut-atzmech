📥 קבלה מאוטומציה (Inbound)

Make / N8N יכולים לשלוח כתבות חדשות ישירות לאדמין דרך URL מיוחד.

1
ב-Make/N8N: הוסיפי מודול HTTP Request שמבצע GET לכתובת הבאה עם פרמטר import.
2
הכתובת שצריך לקרוא לה (החלי את הנתונים ב-Base64 JSON):
admin.html?import=BASE64_JSON
3
ה-JSON צריך להיות מערך: [{...post...}] או אובייקט בודד {...post...} — המערכת תייבא אוטומטית.
4
כאשר הדפדפן ייפתח עם URL זה, הכתבה תתווסף אוטומטית ותקבלי הודעה.
